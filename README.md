# Abils-Mall
Abils Mall is a modern multi‑vendor ecommerce platform with admin/manager dashboards, secure OTP login, delivery mapping with distance‑based shipping, and Paystack/Flutterwave payments and other payment method.

For persistent product images in production (Render), configure Cloudinary env vars: `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET`, `CLOUDINARY_FOLDER`.
